# mi-app

React SPA connected to a JWT REST API.

**Live URL:** `https://mi-app.vercel.app`

---

## Tech Stack

- React 18 + Vite
- React Router DOM v6
- Axios
- Deployed on Vercel

---

## Features

- Register and login with JWT
- Protected routes
- Product CRUD (create, list, delete)

---

## Local Setup

```bash
git clone https://github.com/JoseTomasPM/mi-app.git
cd mi-app
npm install
npm run dev
```

Set your API URL in `src/api/axios.js`:

```js
baseURL: "https://proapi-arwk.onrender.com"
```
